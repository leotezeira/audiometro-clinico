# Android WebView wrapper

Estos archivos son una base mínima para una app Android que abre la web:

- URL: `https://audiometro-clinico.vercel.app/`

## Uso (Android Studio)

1. Crear un proyecto **Empty Views Activity** (Kotlin).
2. Reemplazar los archivos generados por los de:
   - `android/app/src/main/AndroidManifest.xml`
   - `android/app/src/main/java/com/audiometro/clinico/MainActivity.kt`
   - `android/app/src/main/res/layout/activity_main.xml`
3. Asegurarse de que el `package` del proyecto sea `com.audiometro.clinico` o ajustar el `package`/ruta.
